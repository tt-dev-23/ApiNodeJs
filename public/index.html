Api Node JS — api для генерации коротких ссылок к json данным, хранящимся в базе данных.

/generate — POST запрос. На сервер передается json объект, сохраняется в базе, в ответ возвращается уникальная строка: {id}.



Отправить post запрос

/get/{id} — GET запрос. Проверяем, есть ли в базе с таким {id} json объект, если есть то возвращаем его.
